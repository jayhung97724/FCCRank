# FreeCodeCamp

一個以node.js所寫的排行榜，分數來源是<a href="https://www.freecodecamp.com" target="_blank">FreeCodeCamp</a>

## 使用工具
* 前端
  * HTML
  * CSS
  * JavaScript
  * Jade
  * Scss
* 後端
  * Node.js 
  * Express.js
  * Socket.io
* 爬蟲
  * python

## How to start
```
# run python parser
python public/json/fcc.py

npm install -g nodemon
npm install
npm start
```

## 架設環境
* debian 8.4 jessie
* node.js 6.2.1
* python 2.7.12

## Crontab Usage:
* crontab [-l|-e|-r]
	* -l: list all crontab jobs
	* -e: edit crontab jobs
	* -r: remove crontab job
